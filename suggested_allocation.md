## Functional Task Allocation (parallel work streams)

Each member owns one stream beyond their slides. Pick based on strengths:

| Role                                    | Owner | Responsibilities                                                                                    |
| --------------------------------------- | ----- | --------------------------------------------------------------------------------------------------- |
| **Project Lead / Coordinator**          | M1    | Sets meeting cadence, tracks deadlines, owns Moodle submission, manages Miro board structure        |
| **Research Lead — Incident & Timeline** | M2    | Sources Cloudflare post-mortem, FB engineering blog, news timeline; owns reference list             |
| **Technical Lead — RCA**                | M3    | Owns BGP/DNS explanation, builds RCA diagram (Ishikawa or 5 Whys), translates technical → executive |
| **Process & Governance Lead**           | M4    | Researches change mgmt frameworks (ITIL, SRE), maps accountability boundaries, BCP analysis         |
| **Strategy Lead — Recommendations**     | M5    | Drafts risk framework, sources academic/industry refs on resilience, owns trade-off argument        |

## Q&A Preparation (20% of marks!)

Each member becomes the "expert" for likely C-suite questions in their domain:

| Likely Question                                           | Owner           |
| --------------------------------------------------------- | --------------- |
| "Could this happen to us? How likely?"                    | M3 (technical)  |
| "What's the cost-benefit of your recommendations?"        | M5 (strategy)   |
| "Who should be accountable when things like this happen?" | M4 (governance) |
| "How do you balance this with our shipping speed?"        | M5 (trade-offs) |
| "What's the first thing we should do Monday morning?"     | M6 (roadmap)    |
| "Has anything changed at Meta since 2021?"                | M2 (research)   |

**Prep tactic:** run a mock Q&A in final rehearsal — assign one person to play hostile CFO.
